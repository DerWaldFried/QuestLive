# QuestLive

## Überblick

**QuestLive** ist ein Java-Projekt, das darauf abzielt, das tägliche Leben spannender zu gestalten. Es handelt sich um ein minimales System, das jeden Tag **7 Quests** generiert, die man im echten Leben erledigen kann.

## Funktionen

- Generiert täglich 7 einzigartige Quests
- Quests sind auf realistische und erreichbare Ziele ausgerichtet
- Fördert die persönliche Entwicklung und macht das tägliche Leben spannender

## Installation

1. Klonen Sie das Repository: git clone https://github.com/yourusername/QuestLive.git
2. Wechseln sie das Verzeichnis: cd QuestLive
3. Kompilieren Sie das Projekt zum Beispiel mit Maven
4. Führen sie die Anwendung aus

## Nutzung

Nach dem Start der Anwendung generiert QuestLive automatisch 7 Quests für den Tag. Sie können diese Quests anzeigen und auswählen, welche Sie im Laufe des Tages erledigen möchten.
